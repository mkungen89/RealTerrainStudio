# Icon Instructions

## Missing Icon File

The plugin requires an `icon.png` file in this directory for proper display in QGIS.

### Requirements:
- **Filename:** `icon.png`
- **Size:** 24x24 pixels (recommended)
- **Format:** PNG with transparency
- **Content:** Should represent the RealTerrain Studio brand (terrain/mountain icon)

### Temporary Workaround:
The plugin will still load without an icon, but it will show a placeholder or no icon in the QGIS toolbar.

### How to Add:
1. Create or obtain a 24x24 PNG icon
2. Save it as `icon.png` in this directory
3. Reload the plugin in QGIS

### Icon Design Suggestions:
- Mountain/terrain silhouette
- Globe with heightmap
- Topographic contour lines
- UE5 + terrain mashup
- Use colors: Blue/green for terrain, white/gray for elevation
